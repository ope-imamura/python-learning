## 変数とは
### データを保存する「箱」

<span class="importance-high">変数とは、データを一時的に保存しておく「名前付きの箱」のようなもの</span>です。

- コンピュータのメモリ（RAM）上に確保される領域
- 名前（変数名）を付けて、後から参照できる
- 値を保存したり、取り出したり、更新したりできる

<div class="tips">
「変数」という名前は、中身の値が「変わり得るもの」だからです。一度保存した値を、後から別の値に変更することができます。
</div>

---

## 変数とは
### 日常生活の例え

変数をイメージするために、<span class="importance-medium">引き出しや小箱に例えて考えてみましょう</span>。

<div class="columns">
<div>

引き出しの例え：
- 引き出しには名前がついている（変数名）
- 中にモノを入れておける（値の代入）
- あとで取り出せる（値の参照）
- 中身を入れ替えられる（値の更新）

</div>
<div>

<div class="center">
<div class="highlight" style="width: 80%;">
引き出し：「age」<br>
中身：「25」<br><br>
引き出し：「name」<br>
中身：「山田太郎」
</div>
</div>

</div>
</div>

---

## なぜ変数が必要なのか
### 変数の役割と重要性

<span class="importance-high">変数がないと、計算結果や情報を保存しておくことができません。</span>

<div class="columns">
<div>

<span class="importance-medium">変数なしの場合：</span>
```python
# 毎回計算が必要
print(3 + 4)  # 7
print(3 + 4)  # 7
print(3 + 4)  # 7
```

</div>
<div>

<span class="importance-medium">変数ありの場合：</span>
```python
# 一度計算して結果を保存
result = 3 + 4  # 計算して保存
print(result)   # 7
print(result)   # 7
print(result)   # 7
```

</div>
</div>

<div class="tips">
変数を使うことで、一度計算した結果や入力された値を保存し、何度も利用することができます。これにより、同じ処理を何度も繰り返す必要がなくなります。
</div>

---

## なぜ変数が必要なのか
### 変数の主な用途

<span class="importance-high">変数は以下のような場面で活用されます：</span>

<div class="columns">
<div>

- <span class="importance-medium">計算結果の保存</span>
  ```python
  tax_rate = 0.1
  price = 1000
  tax = price * tax_rate
  total = price + tax
  ```

- <span class="importance-medium">入力値の保存</span>
  ```python
  name = input("お名前は？")
  print("こんにちは、" + name + "さん！")
  ```

</div>
<div>

- <span class="importance-medium">状態の追跡</span>
  ```python
  score = 0
  score = score + 10  # 10点追加
  score = score + 5   # さらに5点追加
  ```

- <span class="importance-medium">複雑な値に名前を付ける</span>
  ```python
  pi = 3.14159265359
  radius = 5
  area = pi * radius * radius
  ```

</div>
</div>

## 変数の宣言と代入
### Pythonにおける変数の使い方

<span class="importance-high">Pythonでは、変数に値を代入することで自動的に変数が作成されます。</span>

<div class="columns">
<div>

### 基本的な代入

```python
# 変数名 = 値
age = 25
name = "山田太郎"
is_student = True
height = 175.5
```

</div>
<div>

### 変数の値を使う

```python
age = 25
print(age)  # 25

message = "私は" + str(age) + "歳です"
print(message)  # 私は25歳です
```

</div>
</div>

<div class="tips">
Pythonでは他の言語と異なり、<code>int age;</code>のように事前に変数の宣言をする必要はありません。代入された値に応じて、適切な型が自動的に設定されます。
</div>

---

## 変数の宣言と代入
### 代入の構文と仕組み

<span class="importance-high">代入演算子（=）を使って値を変数に格納します。</span>

<div class="highlight">
<code>変数名 = 値または式</code>
</div>

この処理は次のように行われます：

1. 右側の「値または式」が先に評価される
2. その結果が左側の「変数名」に代入される

```python
x = 10 + 5  # まず 10 + 5 が計算され、結果の 15 が x に代入される

y = x * 2   # まず x の値（15）を取得し、15 * 2 が計算され、結果の 30 が y に代入される
```

---

## 変数の宣言と代入
### 複数の変数の一括代入

<span class="importance-medium">Pythonでは、複数の変数に同時に値を代入することもできます。</span>

```python
# 同じ値を複数の変数に代入
x = y = z = 0
print(x, y, z)  # 0 0 0

# 複数の変数に異なる値を一度に代入
a, b, c = 1, 2, 3
print(a, b, c)  # 1 2 3
```

<div class="tips">
複数の変数に同時代入する方法は便利ですが、コードの可読性が下がる場合もあります。特に初心者の方は、基本的には1行で1つの変数に代入する方法をおすすめします。
</div>

---

## 命名規則
### 変数名の付け方

<span class="importance-high">変数名は自由に付けられますが、いくつかのルールがあります：</span>

<div class="columns">
<div>

### 必ず守るべきルール

- <span class="importance-high">英字、数字、アンダースコア(_)のみ使用可能</span>
- <span class="importance-high">先頭は数字で始められない</span>
- <span class="importance-high">予約語（if, for など）は使えない</span>
- <span class="importance-high">大文字と小文字は区別される</span>（age と AGE は別物）

</div>
<div>

### 推奨される慣習

- <span class="importance-medium">意味のある名前を付ける</span>
- <span class="importance-medium">小文字のスネークケースを使う</span>
  - 例：`user_name`, `total_price`
- <span class="importance-medium">英語を使う</span>
  - 例：`age`（年齢）, `name`（名前）

</div>
</div>

---

## 命名規則
### 良い変数名と悪い変数名の例

<span class="importance-high">わかりやすい変数名を付けることは、プログラムの可読性を高める上で非常に重要です。</span>

<div class="columns">
<div>

### 良い変数名の例

```python
user_age = 25
total_price = 1500
is_logged_in = True
first_name = "太郎"
tax_rate = 0.1
```

</div>
<div>

### 悪い変数名の例

```python
a = 25        # 何の値か不明
tp = 1500     # 略語は避ける
flag = True   # 何のフラグか不明
x1 = "太郎"    # 意味が不明確
t = 0.1       # 短すぎて何か不明
```

</div>
</div>

<div class="tips">
プログラミングの80%は「読む」ことです。数ヶ月後に自分自身が読み返すことを想像して、わかりやすい変数名を心がけましょう。
</div>

---

## 命名規則
### Pythonで使用できない予約語

<span class="importance-medium">以下の予約語は変数名として使用できません：</span>

```
False    await     else      import    pass
None     break     except    in        raise
True     class     finally   is        return
and      continue  for       lambda    try
as       def       from      nonlocal  while
assert   del       global    not       with
async    elif      if        or        yield
```

<div class="tips">
これらの単語はPythonの構文の一部として予約されているため、変数名として使うとエラーになります。
<code>SyntaxError: invalid syntax</code>
または
<code>SyntaxError: cannot assign to ...</code>
のようなエラーが表示されます。
</div>

## 変数の再代入
### 値の更新

<span class="importance-high">変数に代入された値は、後から別の値に変更できます。</span>

```python
# 変数の宣言と初期値の代入
count = 0
print(count)  # 0

# 同じ変数に新しい値を代入（再代入）
count = 10
print(count)  # 10

# さらに別の値に再代入
count = 25
print(count)  # 25
```

<div class="tips">
変数とは「変わり得るもの」という意味があり、このように値を更新できることが変数の重要な特性です。
</div>

---

## 変数の再代入
### 自分自身を使った再代入

<span class="importance-high">変数の現在の値を使って、新しい値を計算し代入することができます。</span>

<div class="columns">
<div>

### カウンターの例

```python
# カウントアップ
score = 0
print(score)  # 0

score = score + 1  # 現在の値に1を加える
print(score)  # 1

score = score + 1  # さらに1を加える
print(score)  # 2
```

</div>
<div>

### 合計の例

```python
# 合計を計算
total = 0
print(total)  # 0

total = total + 100  # 100を加える
print(total)  # 100

total = total + 50   # さらに50を加える
print(total)  # 150
```

</div>
</div>

---

## 変数の再代入

<span class="importance-high">変数への再代入時は、元の値が失われることに注意しましょう。</span>

```python
# 変数初期化
message = "こんにちは"

# 再代入
message = "さようなら"

# 元の値「こんにちは」は失われ、新しい値が格納される
print(message)  # さようなら
```

<div class="tips">
元の値を保持したい場合は、別の変数に保存しておく必要があります。

```python
message1 = "こんにちは"
message2 = message1       # message1の値を別の変数に保存
message1 = "さようなら"    # message1を更新

print(message1)  # さようなら
print(message2)  # こんにちは
```
</div>
